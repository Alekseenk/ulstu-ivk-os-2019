# Лабораторная работа №1

## Цель работы

Изучить понятие виртуализации для операционных систем, произвести установку дистрибутива Linux на виртуальную машину.

## Описание

В рамках этой работы необходимо ознакомиться с понятием виртуализации для операционных систем.
Если кратко, то это возможность запускать операционную систему и программы на ней не на настоящем “железе”, а на эмулируемом.

Плюсы такого подхода:
- в ходе экспериментов с разными операционными системами нельзя испортить уже существующие данные на компьютере, т.к. виртуальная ОС не имеет к ним доступа;
- можно запустить сразу несколько “виртуальных компьютеров” на одном физическом, что иногда полезно для тестирования и разработки;
- при необходимости виртуальную машину можно легко удалить.

Минусы виртуализации для ОС:
- виртуальная машина работает медленнее, чем настоящая (оно и понятно - ведь процессор, память, запоминающие устройства эмулируются);
- иногда возникают проблемы с “перебрасыванием” устройств с физического компьютера в виртуальный. Например, к физическому компьютеру подключена веб-камера.
Необходимо, чтобы виртуальный компьютер тоже мог её использовать. Иногда настроить такую связку трудно. Но реально.

В качестве программы виртуализации рекомендуется использовать VirtualBox (https://www.virtualbox.org/, https://ru.wikipedia.org/wiki/VirtualBox).
Данная программа является крайне популярной: по ней много инструкций, она поддерживает множество операционных систем и интерфейс программы крайне понятен.

В качестве устанавливаемой операционной системы на виртуальный компьютер рекомендуется использовать Ubuntu Linux (https://www.ubuntu.com/, https://ru.wikipedia.org/wiki/Ubuntu).
Эта операционная система активно развивается, у неё удобный интерфейс для обычного пользователя. К тому же, именно этот дистрибутив Linux установлен в лабораториях кафедры ИВК.

## Порядок выполнения работы

1. Установить VirtualBox к себе на компьютер.
2. Настроить VirtualBox для установки Ubuntu Linux.
3. Скачать образ Ubuntu Linux с официального сайта - https://www.ubuntu.com/ (версия ОС – Desktop, LTS).
4. Смонтировать образ Linux к виртуальной машине (как будто вы записали DVD-диск с Linux с вставили его в DVD-ROM компьютера).
5. Произвести установку Linux.
Не бойтесь, виртуальная машина не видит ваши файлы, и ничего не сможет стереть или испортить.
6. _(не обязательно)_ Установите Code::Blocks на Linux и скомпилируйте самую простую программу на языке C.

В качестве справки можно использовать инструкцию с сайта AskUbuntu: https://askubuntu.com/questions/142549/how-to-install-ubuntu-on-virtualbox.

Есть ещё одна неплохая и довольно подробная инструкция по установке Ubuntu: http://profitraders.com/Ubuntu/VirtualBoxUbuntuInstall.html (до пункта “Установка дополнительных программ”).
